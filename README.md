## О проекте

Air Macro — сольный проект, полностью разработанный с нуля.

Программа предназначена для записи, редактирования и воспроизведения макросов с высокой точностью таймингов и поддержкой логических условий.

В отличие от типичных макросников периферии, используется собственный движок воспроизведения и низкоуровневый ввод через WinAPI.

---

## Ключевые особенности

### Движок ввода
- SendInput (аппаратный уровень)
- WH_MOUSE_LL + WH_KEYBOARD_LL
- Абсолютные координаты мыши
- timeBeginPeriod(1) при необходимости

### Точность
- Задержки от ~1 мс
- Стабильная работа при 1000+ действиях
- Скорость воспроизведения 0.02× – 50×

### Условия и логика
- WaitPixel (ожидание цвета)
- IfPixelColor (ветвление)
- Проверка цвета в реальном времени

### Рандомизация
- Задержки: 0–999 мс
- Смещение клика: ±0–99 px

### Спамер
- Интервал 1–1000 мс
- Отдельный поток
- Поддержка XButton1 / XButton2

### Редактор
- Табличный интерфейс (DataGridView)
- Копирование / вставка / дублирование
- Массовое изменение задержек
- Импорт / экспорт JSON

### Инструменты
- Пипетка цвета
- HEX / RGB / координаты
- Палитра часто используемых цветов
